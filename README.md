# Discord Emoji Manager

Discordのカスタム絵文字をカテゴリ分け・ソートできるChrome拡張機能。

## 機能

- **カテゴリ管理** — 好きなフォルダを作って絵文字を整理
- **ソート** — 名前順・逆順・サーバー順
- **検索** — 名前でフィルター
- **まとめて移動** — 複数選択してカテゴリを一括変更（長押し or Shiftクリック）
- **直接挿入** — クリックするとメッセージ入力欄に挿入
- **ドラッグ移動** — パネルを好きな位置に移動可
- **設定を保存** — カテゴリ・割り当てはブラウザに永続保存

## インストール方法

1. このフォルダ（`discord-emoji-manager`）をダウンロード
2. Chromeを開いて `chrome://extensions` へ
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択 → 完了

## 使い方

1. Discordを開く
2. ページ右下に ⭐ ボタンが出るのでクリック
3. **絵文字を読み込む**: Discord の絵文字ピッカー（😊ボタン）を一度開く
   → 自動で絵文字が取り込まれます
4. パネルで絵文字をクリック → メッセージ欄に挿入
5. カテゴリを追加して整理！

## 技術メモ

### 絵文字の取得について
Discord は絵文字データをWebpack内部に持っているため、
公式APIを使わずにDOMから収集する方式を採用しています。
絵文字ピッカーを開いたときに表示されている絵文字を読み取ります。

### データの保存
`chrome.storage.local` に保存されます（ブラウザに紐付け）。
Discordのサーバーには一切送信しません。

## ファイル構成

```
discord-emoji-manager/
├── manifest.json       # 拡張機能の設定
├── src/
│   ├── content.js      # メインスクリプト（DOMに注入）
│   └── styles.css      # パネルのスタイル
└── icons/              # アイコン画像（別途用意）
```

## 注意

- Discordのアップデートでセレクタが変わると動かなくなる場合があります
- カスタム絵文字のみ対応（Unicode絵文字は対象外）
- Discordの利用規約の範囲内での利用をお願いします
